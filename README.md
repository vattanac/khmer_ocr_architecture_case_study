# Khmer OCR Architecture — Case Study

A personal research reference project exploring state-of-the-art Optical Character Recognition (OCR) architectures applied to the Khmer script. This repository consolidates findings from academic papers and expert research for study and reference purposes.

> **Note:** This project is intended for personal academic use only — it is a curated study of existing research, not an original model or production system.

---

## 📚 Overview

Khmer script presents unique challenges for OCR systems due to its complex character stacking, ligatures, and diacritics. This case study reviews and visualizes architectural approaches from published research to better understand how modern OCR pipelines handle low-resource, morphologically complex scripts.
